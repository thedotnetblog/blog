Bookmark Studio добавляет слотовую навигацию и обмен закладками в Visual Studio

Новое расширение Bookmark Studio от Mads Kristensen добавляет навигацию по слотам с клавиатуры, менеджер закладок, цвета, метки и возможности экспорта к закладкам Visual Studio.

Закладки в Visual Studio всегда были… нормальными. Ставишь одну, переходишь к следующей, забываешь какая есть какая. Работают, но никогда не были чем-то, что назвали бы мощным.

Mads Kristensen только что выпустил Bookmark Studio — экспериментальное расширение, которое заполняет именно те пробелы, с которыми вы, вероятно, сталкивались.

Слотовая навигация

Закладки можно назначить на слоты 1–9 и перейти к ним напрямую через Alt+Shift+1 по Alt+Shift+9. Новые закладки автоматически получают следующий свободный слот.

Менеджер закладок

Новое окно инструментов показывает все закладки с фильтрацией по имени, файлу, расположению, цвету или слоту.

Организация с метками, цветами и папками

Закладки могут иметь метки, цвета и группироваться в папки. Метаданные сохраняются по solution.

Экспорт и обмен

Bookmark Studio позволяет экспортировать закладки как текст, Markdown или CSV.

Закладки, следующие за кодом

Bookmark Studio отслеживает закладки относительно привязанного текста, поэтому они не сдвигаются на неправильные строки при редактировании.

Подведём итоги

Bookmark Studio ничего не переизобретает. Берёт функцию, которая годами была «достаточной», и делает её действительно полезной. Скачайте из Visual Studio Marketplace.
