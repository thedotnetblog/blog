Пишите для The .NET Blog

Делитесь своими знаниями с .NET-сообществом.

The .NET Blog — это публикация, управляемая сообществом, где разработчики делятся инсайтами, туториалами и историями о .NET, Azure, ИИ и облачно-нативной разработке. Мы приветствуем вклад разработчиков любого уровня — будь вы новичком, пишущим первую техническую статью, или опытным докладчиком.

Как присоединиться

Всё хранится на GitHub и следует рабочему процессу pull request. Вот как начать:

1. Форкните репозиторий

Перейдите на github.com/thedotnetblog/blog и форкните репозиторий в свой аккаунт GitHub.

2. Создайте профиль автора

Создайте папку content/authors/your-username/ с файлом index.md, указав имя, роль и биографию. Добавьте изображение аватара (квадратное, минимум 200×200px) в static/img/authors/.

3. Напишите статью

Создайте папку content/posts/your-username/your-post-slug/ и добавьте файл index.md с заголовком, датой, автором, описанием и тегами.

4. Откройте Pull Request

Отправьте изменения в ваш форк и откройте pull request в ветку main. Наша команда проверит его и даст обратную связь в течение нескольких дней.

Что мы ищем

  • Глубокий анализ — детальное изучение технологий, паттернов или архитектур

Рекомендации

  • Контент должен быть техническим и релевантным для экосистемы .NET
  • Примеры кода должны быть точными и протестированными в реальном проекте
  • Включите осмысленное описание и хотя бы один релевантный тег
  • Статьи автоматически переводятся на все поддерживаемые языки
