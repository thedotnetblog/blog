Global Azure Spain 2026 a lieu le **18 avril 2026** au **Kinépolis Diversia** à Alcobendas, Madrid. C'est le plus grand événement communautaire Azure en Espagne, réunissant 38 speakers sur 3 tracks parallèles couvrant les agents IA, le réseau Azure, Cosmos DB, Fabric, IoT, la sécurité et bien plus.

L'événement se déroule de **08h30 à 18h30** et comprend une keynote, des pauses café, un déjeuner et une session de clôture Q&A.

## Points forts de l'agenda

- **Domando Agentes de IA** : gouvernance, outils et APIs avec Azure AI Foundry et Azure API Management
- **Construyendo agentes con LibreChat en Azure**
- **How Can I Steal Your Data with Azure Private Endpoints**
- **Stop Building APIs. Forge Agents with Azure**
- **Agentic DevOps Meets IoT: Real-Time Systems with Fabric and GitHub Copilot**
- **El regreso de los tamagotchis!** : systèmes multi-agents en action
- **Foundry Control Plane como plataforma de Agentes global**
- **Rompiendo el perímetro: Zero Trust aplicado en Azure**
