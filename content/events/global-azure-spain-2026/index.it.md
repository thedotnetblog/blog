Global Azure Spain 2026 si tiene il **18 aprile 2026** al **Kinépolis Diversia** ad Alcobendas, Madrid. È il più grande evento comunitario Azure in Spagna, con 38 speaker su 3 track paralleli dedicati ad agenti IA, networking Azure, Cosmos DB, Fabric, IoT, sicurezza e molto altro.

L'evento si svolge dalle **08:30 alle 18:30** e include keynote, pause caffè, pranzo e una sessione di chiusura Q&A.

## Punti salienti dell'agenda

- **Domando Agentes de IA**: governance, strumenti e API con Azure AI Foundry e Azure API Management
- **Construyendo agentes con LibreChat en Azure**
- **How Can I Steal Your Data with Azure Private Endpoints**
- **Stop Building APIs. Forge Agents with Azure**
- **Agentic DevOps Meets IoT: Real-Time Systems with Fabric and GitHub Copilot**
- **El regreso de los tamagotchis!**: sistemi multi-agente in azione
- **Foundry Control Plane como plataforma de Agentes global**
- **Rompiendo el perímetro: Zero Trust aplicado en Azure**
