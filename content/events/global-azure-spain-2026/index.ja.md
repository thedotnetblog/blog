Global Azure Spain 2026 は **2026年4月18日** にマドリード、Alcobendas の **Kinépolis Diversia** で開催されます。スペイン最大のコミュニティ主導のAzureイベントで、AIエージェント、Azureネットワーク、Cosmos DB、Fabric、IoT、セキュリティなどをカバーする3つの並行トラックに38名のスピーカーが登壇します。

イベントは **08:30 から 18:30** まで開催され、キーノート、コーヒーブレイク、ランチ、クロージングQ&Aセッションを含みます。

## アジェンダのハイライト

- **Domando Agentes de IA**: Azure AI FoundryとAzure API Managementによるガバナンス、ツール、API
- **Construyendo agentes con LibreChat en Azure**
- **How Can I Steal Your Data with Azure Private Endpoints**
- **Stop Building APIs. Forge Agents with Azure**
- **Agentic DevOps Meets IoT: Real-Time Systems with Fabric and GitHub Copilot**
- **El regreso de los tamagotchis!**: マルチエージェントシステムの実践
- **Foundry Control Plane como plataforma de Agentes global**
- **Rompiendo el perímetro: Zero Trust aplicado en Azure**
