The .NET Blog는 개발자들이 .NET, Azure, AI, 클라우드 네이티브 개발에 대한 인사이트, 튜토리얼, 이야기를 공유하는 커뮤니티 기반 출판물입니다. **모든 수준의 개발자의 기고를 환영합니다** — 첫 번째 기술 글을 쓰는 분이든, 경험 많은 연사든 모두 환영합니다.

## 참여 방법

모든 것은 GitHub에 있으며 풀 리퀘스트 워크플로우를 따릅니다.

### 1. 저장소 포크하기

[github.com/thedotnetblog/blog](https://github.com/thedotnetblog/blog)로 이동하여 GitHub 계정으로 포크하세요.

### 2. 저자 프로필 만들기

`content/authors/your-username/` 폴더를 만들고 `index.md` 파일을 추가하세요. 이름, 직책, 소개를 포함하고 아바타 이미지(정사각형, 최소 200×200px)를 `static/img/authors/`에 추가하세요.

### 3. 글 작성하기

`content/posts/your-username/your-post-slug/`에 새 폴더를 만들고 `index.md` 파일을 추가하세요. 제목, 날짜, 저자, 설명, 태그를 포함하세요.

### 4. 풀 리퀘스트 열기

변경 사항을 포크에 푸시하고 `main` 브랜치에 풀 리퀘스트를 여세요. 팀이 며칠 내에 검토하고 피드백을 드립니다.

## 찾고 있는 콘텐츠

- **튜토리얼** — .NET, Azure, AI, Blazor, Aspire 등에 대한 단계별 가이드
- **심층 분석** — 기술, 패턴 또는 아키텍처에 대한 상세한 탐구
- **커뮤니티 이야기** — 프로덕션에서 .NET을 사용한 경험
- **이벤트 요약** — 컨퍼런스, 밋업 또는 웨비나 요약

## 가이드라인

- 콘텐츠는 .NET 생태계와 관련된 기술적인 내용이어야 합니다
- 코드 예제는 실제 프로젝트에서 정확하고 테스트된 것이어야 합니다
- 의미 있는 설명과 관련 태그를 최소 하나 이상 포함하세요
- 글은 지원되는 모든 언어로 자동 번역됩니다
