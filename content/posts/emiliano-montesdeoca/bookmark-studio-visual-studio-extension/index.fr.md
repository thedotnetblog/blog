Les signets dans Visual Studio ont toujours été... corrects. Vous en placez un, vous naviguez au suivant, vous oubliez lequel est lequel. Ça fonctionne, mais ce n'a jamais été une fonctionnalité qu'on qualifierait de puissante.

Mads Kristensen vient de [sortir Bookmark Studio](https://devblogs.microsoft.com/visualstudio/bookmark-studio-evolving-bookmarks-in-visual-studio/), une extension expérimentale qui comble exactement les lacunes que vous avez probablement rencontrées.

## Navigation par slots

Les signets peuvent être assignés aux slots 1–9 et accessibles directement avec `Alt+Shift+1` à `Alt+Shift+9`. Les nouveaux signets obtiennent automatiquement le prochain slot disponible.

## Le Gestionnaire de Signets

Une nouvelle fenêtre d'outils affiche tous vos signets avec filtrage par nom, fichier, emplacement, couleur ou slot.

## Organisation avec labels, couleurs et dossiers

Les signets peuvent avoir des labels, couleurs et être groupés en dossiers. Toutes les métadonnées sont stockées par solution.

## Exporter et partager

Bookmark Studio permet d'exporter les signets en texte brut, Markdown ou CSV.

## Signets qui suivent le code

Bookmark Studio suit les signets relativement au texte auquel ils sont ancrés, ils ne dérivent donc pas vers les mauvaises lignes.

## Pour conclure

Bookmark Studio ne réinvente rien. Il prend une fonctionnalité qui était « suffisante » depuis des années et la rend vraiment utile. Téléchargez-le depuis le [Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=MadsKristensen.BookmarkStudio).
