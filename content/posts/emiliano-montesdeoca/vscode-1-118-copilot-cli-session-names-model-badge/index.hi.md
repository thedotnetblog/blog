*यह पोस्ट स्वचालित रूप से अनुवादित है। मूल संस्करण के लिए [यहाँ क्लिक करें]({{< ref "index.md" >}})।*

[Visual Studio Code 1.118](https://code.visualstudio.com/updates/v1_118) एक छोटा, केंद्रित रिलीज़ है — मुख्यतः Copilot CLI सुधार।

## Copilot CLI: सेशन को असली नाम मिलते हैं

Copilot CLI SDK के सेशन टाइटल APIs अब सेशन नामों के स्रोत के रूप में उपयोग होते हैं। ऑटो-जेनरेटेड लेबल की बजाय SDK का असली नाम दिखता है।

## कीबोर्ड शॉर्टकट से तेज़ सेशन स्विचिंग

Agents ऐप में अब `Ctrl+1`, `Ctrl+2` आदि से सेशन जल्दी बदले जा सकते हैं।

## चैट में मॉडल बैज

Copilot CLI की चैट पैनल में प्रतिक्रियाओं पर अब मॉडल बैज दिखता है।

## Copilot CLI में ऑटो मॉडल सिलेक्शन

ऑटो मॉडल सिलेक्शन अब Copilot CLI एजेंट में भी काम करता है।

## TypeScript 7.0 Nightly ऑप्ट-इन

VS Code सेटिंग्स से सीधे TypeScript 7.0 नाइटली टेस्ट करने का विकल्प मिलता है। TypeScript 7.0 एक बड़ा रिलीज़ है ([बीटा कुछ दिन पहले आया](https://devblogs.microsoft.com/typescript/announcing-typescript-7-0-beta/))।

[पूर्ण रिलीज़ नोट्स](https://code.visualstudio.com/updates/v1_118) देखें।
