[Visual Studio Code 1.118](https://code.visualstudio.com/updates/v1_118) è una release più piccola e focalizzata — principalmente raffinamenti di Copilot CLI.

## Copilot CLI: le sessioni hanno nomi reali

Le API di titolo di sessione dell'SDK di Copilot CLI vengono ora usate come fonte di verità per i nomi delle sessioni. Invece di etichette auto-generate, le sessioni mostrano il nome reale dall'SDK.

## Cambia sessione più velocemente

L'app Agents ha ora `Ctrl+1`, `Ctrl+2`, ecc. per passare rapidamente tra le sessioni.

## Badge modello nella chat

Le risposte di Copilot CLI nel pannello chat mostrano ora un badge modello — puoi vedere a colpo d'occhio quale modello ha gestito ogni richiesta.

## Selezione automatica del modello in Copilot CLI

La selezione automatica del modello funziona ora anche nell'agente Copilot CLI.

## Opt-in per TypeScript 7.0 Nightly

Puoi ora optare per testare i nightly di TypeScript 7.0 direttamente dalle impostazioni di VS Code. TypeScript 7.0 è un release importante (la [beta è uscita pochi giorni fa](https://devblogs.microsoft.com/typescript/announcing-typescript-7-0-beta/)).

Consulta le [note di release complete](https://code.visualstudio.com/updates/v1_118).
