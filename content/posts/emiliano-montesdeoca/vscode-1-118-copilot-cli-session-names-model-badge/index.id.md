[Visual Studio Code 1.118](https://code.visualstudio.com/updates/v1_118) adalah rilis kecil yang terfokus — sebagian besar penyempurnaan Copilot CLI.

## Copilot CLI: Sesi Mendapat Nama Nyata

API judul sesi SDK Copilot CLI kini digunakan sebagai sumber kebenaran untuk nama sesi. Alih-alih label yang dibuat otomatis, sesi menampilkan nama asli dari SDK.

## Beralih Sesi Lebih Cepat

Aplikasi Agents kini memiliki `Ctrl+1`, `Ctrl+2`, dll. untuk beralih cepat antar sesi.

## Lencana Model di Chat

Respons Copilot CLI di panel chat kini menampilkan lencana model — Anda bisa melihat sekilas model mana yang menangani setiap permintaan.

## Pemilihan Model Otomatis di Copilot CLI

Fitur pemilihan model otomatis kini juga berfungsi di agen Copilot CLI.

## Opt-in TypeScript 7.0 Nightly

Anda kini bisa memilih untuk menguji TypeScript 7.0 nightly langsung dari pengaturan VS Code. TypeScript 7.0 adalah rilis besar ([beta dirilis beberapa hari lalu](https://devblogs.microsoft.com/typescript/announcing-typescript-7-0-beta/)).

Lihat [catatan rilis lengkap](https://code.visualstudio.com/updates/v1_118).
