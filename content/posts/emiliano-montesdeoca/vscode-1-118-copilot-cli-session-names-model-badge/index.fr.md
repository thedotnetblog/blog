[Visual Studio Code 1.118](https://code.visualstudio.com/updates/v1_118) est une version plus petite et ciblée — principalement des raffinements de Copilot CLI — mais quelques éléments méritent d'être notés.

## Copilot CLI : les sessions ont de vrais noms

Les APIs de titre de session du SDK Copilot CLI servent maintenant de source de vérité pour les noms de session. Les sessions affichent désormais le vrai nom du SDK au lieu d'étiquettes générées automatiquement.

## Changer de session plus rapidement

L'application Agents a maintenant `Ctrl+1`, `Ctrl+2`, etc. pour basculer rapidement entre les sessions.

## Badges de modèle dans le chat

Les réponses Copilot CLI dans le panneau de chat affichent maintenant un badge de modèle pour voir quel modèle a traité chaque requête.

## Sélection automatique de modèle dans Copilot CLI

La sélection automatique de modèle fonctionne maintenant aussi dans l'agent Copilot CLI.

## Opt-in TypeScript 7.0 Nightly

Vous pouvez maintenant tester les nightlies TypeScript 7.0 directement depuis les paramètres VS Code. TypeScript 7.0 est une version majeure (la [bêta est sortie il y a quelques jours](https://devblogs.microsoft.com/typescript/announcing-typescript-7-0-beta/)).

Consultez les [notes de version complètes](https://code.visualstudio.com/updates/v1_118).
