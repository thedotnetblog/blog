[Visual Studio Code 1.118](https://code.visualstudio.com/updates/v1_118), ağırlıklı olarak Copilot CLI iyileştirmelerine odaklanan küçük bir sürümdür.

## Copilot CLI: Oturumlara Gerçek Adlar Geliyor

Copilot CLI SDK oturum başlığı API'leri artık oturum adları için doğruluk kaynağı olarak kullanılıyor. Otomatik oluşturulan etiketler yerine SDK'daki gerçek ad gösteriliyor.

## Klavye Kısayollarıyla Hızlı Oturum Değiştirme

Agents uygulamasına artık oturumlar arasında hızla geçiş için `Ctrl+1`, `Ctrl+2` vb. tuş atamaları eklendi.

## Sohbette Model Rozetleri

Copilot CLI'nin sohbet panelindeki yanıtlarında artık model rozeti gösteriliyor.

## Copilot CLI'da Otomatik Model Seçimi

Otomatik model seçimi özelliği artık Copilot CLI ajanında da çalışıyor.

## TypeScript 7.0 Nightly Katılımı

Artık VS Code ayarlarından doğrudan TypeScript 7.0 nightly sürümlerini test etmeyi etkinleştirebilirsiniz. TypeScript 7.0 önemli bir sürümdür ([beta birkaç gün önce yayınlandı](https://devblogs.microsoft.com/typescript/announcing-typescript-7-0-beta/)).

[Tam sürüm notlarına](https://code.visualstudio.com/updates/v1_118) bakın.
