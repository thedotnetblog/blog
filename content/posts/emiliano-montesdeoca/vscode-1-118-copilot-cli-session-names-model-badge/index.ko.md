[Visual Studio Code 1.118](https://code.visualstudio.com/updates/v1_118)은 작지만 집중된 릴리스입니다 — 주로 Copilot CLI 개선에 관한 것이지만 몇 가지 주목할 점이 있습니다.

## Copilot CLI: 세션에 진짜 이름 붙기

Copilot CLI SDK 세션 타이틀 API가 이제 세션 이름의 진실 소스로 사용됩니다. 자동 생성된 레이블 대신 SDK의 실제 이름이 표시됩니다.

## 키보드 단축키로 세션 빠르게 전환

Agents 앱에 이제 `Ctrl+1`, `Ctrl+2` 등의 세션 전환 키가 바인딩되었습니다.

## 채팅에 모델 배지 표시

채팅 패널의 Copilot CLI 응답에 이제 모델 배지가 표시됩니다 — 각 요청을 어떤 모델이 처리했는지 한눈에 볼 수 있습니다.

## Copilot CLI에 자동 모델 선택

자동 모델 선택 기능이 이제 Copilot CLI 에이전트에서도 작동합니다.

## TypeScript 7.0 나이틀리 옵트인

이제 VS Code 설정에서 직접 TypeScript 7.0 나이틀리 테스트에 옵트인할 수 있습니다. TypeScript 7.0은 대규모 릴리스입니다([베타가 며칠 전 출시](https://devblogs.microsoft.com/typescript/announcing-typescript-7-0-beta/)).

[전체 릴리스 노트](https://code.visualstudio.com/updates/v1_118)를 확인하세요.
