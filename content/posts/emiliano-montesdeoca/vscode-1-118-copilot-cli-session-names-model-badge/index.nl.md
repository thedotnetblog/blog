[Visual Studio Code 1.118](https://code.visualstudio.com/updates/v1_118) is een kleinere, gerichte release — voornamelijk Copilot CLI-verbeteringen.

## Copilot CLI: Sessies Krijgen Echte Namen

De sessietitel-API's van de Copilot CLI SDK worden nu gebruikt als bron van waarheid voor sessienamen. In plaats van automatisch gegenereerde labels tonen sessies nu de echte naam uit de SDK.

## Sneller Wisselen Tussen Sessies

De Agents-app heeft nu `Ctrl+1`, `Ctrl+2`, enz. voor snel wisselen tussen sessies.

## Modelbadges in de Chat

Copilot CLI-antwoorden in het chatpaneel tonen nu een modelbadge — je ziet in één oogopslag welk model elk verzoek heeft afgehandeld.

## Automatische Modelselectie in Copilot CLI

De automatische modelselectie werkt nu ook in de Copilot CLI-agent.

## TypeScript 7.0 Nightly Opt-in

Je kunt nu rechtstreeks vanuit de VS Code-instellingen kiezen voor het testen van TypeScript 7.0-nightlies. TypeScript 7.0 is een grote release ([de beta verscheen een paar dagen geleden](https://devblogs.microsoft.com/typescript/announcing-typescript-7-0-beta/)).

Bekijk de [volledige release-opmerkingen](https://code.visualstudio.com/updates/v1_118).
