[Visual Studio Code 1.118](https://code.visualstudio.com/updates/v1_118) és un llançament petit i enfocat — principalment millores de Copilot CLI.

## Copilot CLI: les sessions ara tenen noms reals

Les APIs de títol de sessió del SDK de Copilot CLI s'utilitzen ara com a font de veritat. Les sessions mostren el nom real del SDK en lloc d'etiquetes auto-generades.

## Canvia de sessió amb dreceres de teclat

L'app d'Agents ara té `Ctrl+1`, `Ctrl+2`, etc. per canviar ràpidament entre sessions.

## Insígnies de model al xat

Les respostes de Copilot CLI al panell de xat ara mostren una insígnia de model per veure quin model ha gestionat cada sol·licitud.

## Selecció automàtica de model a Copilot CLI

La selecció automàtica de model — anteriorment disponible en altres parts de Copilot — ara funciona també a l'agent Copilot CLI.

## Opt-in a TypeScript 7.0 nightly

Ara pots optar per provar els nightly de TypeScript 7.0 directament des de la configuració de VS Code.

Consulta les [notes de versió completes](https://code.visualstudio.com/updates/v1_118).
