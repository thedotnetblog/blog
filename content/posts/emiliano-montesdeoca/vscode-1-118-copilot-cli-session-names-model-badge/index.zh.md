[Visual Studio Code 1.118](https://code.visualstudio.com/updates/v1_118) 是一个较小的专注版本——主要是 Copilot CLI 的改进——但有几点值得关注。

## Copilot CLI：会话拥有真实名称

Copilot CLI SDK 的会话标题 API 现在用作会话名称的真实来源。会话显示 SDK 中的实际名称，而非自动生成的标签。

## 使用键盘快捷键快速切换会话

Agents 应用现在绑定了 `Ctrl+1`、`Ctrl+2` 等快捷键，用于快速切换会话。

## 聊天中的模型徽章

Copilot CLI 在聊天面板中的回复现在显示模型徽章——一眼即可看出哪个模型处理了每个请求。

## Copilot CLI 中的自动模型选择

自动模型选择功能——之前在 Copilot 的其他部分已可用——现在也在 Copilot CLI 代理中工作。

## TypeScript 7.0 夜间版选择加入

现在可以直接从 VS Code 设置中选择加入测试 TypeScript 7.0 夜间版。TypeScript 7.0 是一个重大版本（[测试版几天前发布](https://devblogs.microsoft.com/typescript/announcing-typescript-7-0-beta/)）。

查看[完整发行说明](https://code.visualstudio.com/updates/v1_118)。
