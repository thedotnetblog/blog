## Foundry Agent Service siap untuk produksi

Runtime agen generasi berikutnya tersedia secara umum — dibangun di atas OpenAI Responses API, kompatibel secara wire dengan agen OpenAI.

Penambahan utama: jaringan privat end-to-end, ekspansi auth MCP, pratinjau Voice Live, dan hosted agents di 6 region baru.
