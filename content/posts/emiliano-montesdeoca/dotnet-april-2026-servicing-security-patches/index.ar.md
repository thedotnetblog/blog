[تحديثات الصيانة لأبريل 2026](https://devblogs.microsoft.com/dotnet/dotnet-and-dotnet-framework-april-2026-servicing-updates/) لـ .NET و.NET Framework متاحة، وتتضمن إصلاحات أمنية تريد تطبيقها قريباً. ستة CVEs مُصلَّحة، بما في ذلك نقطتا ضعف لتنفيذ التعليمات البرمجية عن بُعد (RCE).

## ما تم تصحيحه

| CVE | النوع | يؤثر على |
|-----|------|---------|
| CVE-2026-26171 | تجاوز ميزة الأمان | .NET 10, 9, 8 + .NET Framework |
| CVE-2026-32178 | **تنفيذ التعليمات البرمجية عن بُعد** | .NET 10, 9, 8 + .NET Framework |
| CVE-2026-33116 | **تنفيذ التعليمات البرمجية عن بُعد** | .NET 10, 9, 8 |
| CVE-2026-32203 | رفض الخدمة | .NET 10, 9, 8 + .NET Framework |
| CVE-2026-23666 | رفض الخدمة | .NET Framework 3.0–4.8.1 |
| CVE-2026-32226 | رفض الخدمة | .NET Framework 2.0–4.8.1 |

## الإصدارات المُحدَّثة

- **.NET 10**: 10.0.6
- **.NET 9**: 9.0.15
- **.NET 8**: 8.0.26

## ما يجب فعله

حدّث مشاريعك وخطوط CI/CD إلى أحدث إصدارات التصحيح. نقطتا ضعف RCE ليستا شيئاً تؤجله.
