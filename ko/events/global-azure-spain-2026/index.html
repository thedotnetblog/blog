Global Azure Spain 2026

Global Azure Spain 2026은 2026년 4월 18일 마드리드 Alcobendas의 Kinépolis Diversia에서 개최됩니다. 스페인 최대의 커뮤니티 주도 Azure 이벤트로, AI 에이전트, Azure 네트워킹, Cosmos DB, Fabric, IoT, 보안 등을 다루는 3개 병렬 트랙에 38명의 발표자가 참여합니다.

이벤트는 08:30부터 18:30까지 진행되며, 키노트, 커피 브레이크, 점심, 클로징 Q&A 세션을 포함합니다.
