.NET 10이 Ubuntu 26.04 LTS에 탑재됐다 — 새로운 것들

Ubuntu 26.04 LTS (Resolute Raccoon)가 .NET 10을 일급 도구체인으로 탑재했습니다. Native AOT, Chiseled 컨테이너, Linux 7.0.

Ubuntu LTS의 날입니다. Ubuntu 26.04 (Resolute Raccoon)이 오늘 출시됐으며, 모든 Ubuntu LTS처럼 최신 .NET LTS — 이번에는 .NET 10 — 을 탑재합니다.

.NET 10을 두 명령으로 설치

sudo apt update
sudo apt install dotnet-sdk-10.0

.NET은 Ubuntu에서 공식 지원하는 도구체인 중 하나입니다.

컨테이너: -noble을 -resolute로 업데이트

sed -i "s/noble/resolute/g" Dockerfile

Native AOT: 3ms 시작, 1.4MB 바이너리

apt install -y dotnet-sdk-aot-10.0 clang
dotnet publish app.cs
# 1.4MB 네이티브 바이너리, 3ms 시작

콜드 스타트 시간이 중요한 클라우드 네이티브 워크로드 — Functions, 컨테이너, 서버리스 — 에서 진정한 게임 체인저입니다.

.NET 8 또는 9가 필요한가요?

apt install -y software-properties-common
add-apt-repository ppa:dotnet/backports
apt install -y dotnet-sdk-8.0

전체 게시물에서 cgroup v2, 포스트-퀀텀 암호화, Linux 7.0에 대한 자세한 내용을 확인하세요.
