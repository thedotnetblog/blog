The .NET Blog に投稿する

.NET コミュニティにあなたの知識を共有しよう。

The .NET Blog は、開発者が .NET、Azure、AI、クラウドネイティブ開発に関する洞察、チュートリアル、ストーリーを共有するコミュニティ主導の出版物です。すべてのレベルの開発者からの投稿を歓迎します — 初めての技術記事を書く方も、経験豊富なスピーカーも大歓迎です。

参加方法

すべては GitHub 上にあり、プルリクエストのワークフローに従います。

1. リポジトリをフォークする

github.com/thedotnetblog/blog にアクセスし、あなたの GitHub アカウントにフォークしてください。

2. 著者プロフィールを作成する

初めての投稿の場合、content/authors/your-username/ フォルダを作成し、index.md ファイルに名前、役割、自己紹介を追加してください。アバター画像（正方形、最低 200×200px）を static/img/authors/ に追加してください。

3. 記事を書く

content/posts/your-username/your-post-slug/ に新しいフォルダを作成し、タイトル、日付、著者、説明、タグを含む index.md ファイルを追加してください。

4. プルリクエストを開く

変更をあなたのフォークにプッシュし、main ブランチへのプルリクエストを開いてください。チームが数日以内にレビューしてフィードバックを提供します。

求めているコンテンツ

  • チュートリアル — .NET、Azure、AI、Blazor、Aspire などのステップバイステップガイド
  • ディープダイブ — 技術、パターン、アーキテクチャの詳細な解説
  • コミュニティストーリー — 本番環境での .NET 活用経験
  • イベントレポート — カンファレンス、ミートアップ、ウェビナーのまとめ

ガイドライン

  • コンテンツは .NET エコシステムに関連する技術的なものであること
  • コード例は実際のプロジェクトで動作確認済みであること
  • 意味のある説明と少なくとも 1 つの関連タグを含めること
  • 記事はサポートされているすべての言語に自動翻訳されます

お問い合わせ
